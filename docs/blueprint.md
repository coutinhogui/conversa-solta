# **App Name**: Shuffle Talks PWA

## Core Features:

- Home Screen Carousel: Displays a carousel of featured card decks with quick actions like 'Start Conversation'.
- Deck Listing and Filtering: Lists all available card decks, allowing users to filter by tags or category, and favorite/unfavorite decks.
- Conversation Session: Initiates a conversation session with shuffled questions from a selected deck, providing controls for next, skip, and reset.
- Favorite Decks: Lists and allows reordering of favorited card decks, with an option to remove them.
- Settings: Allows users to toggle between light and dark themes and view connectivity status and offline instructions.
- Web Share API Integration: Enables users to share decks/links via Web Share API, falling back to copying the link if the API is unavailable.
- Firestore Sync: Uses Firestore for syncing decks' content and metadata to enable offline functionality using only free tier.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a calm and engaging atmosphere.
- Background color: Light Gray (#F5F5F5) for a clean, unobtrusive backdrop.
- Accent color: Amber (#FFC107) for CTAs and important UI elements to draw user attention.
- Body and headline font: 'PT Sans', a versatile and modern sans-serif, suitable for all text elements in the application.
- Code font: 'Source Code Pro' for displaying code snippets.
- Minimalistic, flat icons with subtle color highlights corresponding to the brand's primary and accent colors.
- Subtle animations and transitions to enhance user experience without being distracting.
